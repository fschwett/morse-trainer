## Concept:

The Koch method is a widely used method to learn Morse code. It's based on the idea of hearing the code at the full speed to learn the actual rhythm. The conventional method was to just memorize the dots and dashes but this leads to learning plateaus when building speed. The Koch methods avoids this. When practicing, you hear five characters in cw and try to immediatly write them down. This way it's closer to learning a language instead of a code and you can actually get fluent quickly. Also not all characters are practiced from the beginning. Instead you start with only two or three characters and build your way up.

I wanted to have a website that gives me full controll over which characters, which speed and so on I wanted to practice. I ended up implementing three different modes - continuous, block, and test. All of them let you choose the characters, the speed and the pitch and all of them give you the option to mute the tone, show the code in form of a flashing light and to show the code directly as dots and dashes.

* **Continous** just keeps playing Morse code at the speed and pitch you chose.
* **Block** is the same, except it always plays five characters and then waits for a specified amount of time before revealing the code.
* **Test** plays a block of five characters. The code has to be typed into a input and is then checked.
